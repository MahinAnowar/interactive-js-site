Question-1: What are the different ways to select an element in the DOM?

I can select elements in the DOM using methods like document.getElementById() to pick one by its ID, document.querySelector() to grab the first match of a CSS selector or document.getElementsByClassName() to get all elements with a specific class. There is also document.getElementsByTagName() to select all elements of a certain tag, like all tags.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML lets you set or get the HTML code inside an element. including tags. But innerText just deals with the visible text you see on the page. textContent is similar to innerText but grabs all the text even if it's hidden by CSS or inside script tags.

Question-3: What is event delegation in the DOM?

Event delegation is when you add an event listener to a parent element instead of each child element. The event then bubbles up so you can handle multiple child elements with one event listener.

Question-4: What is event bubbling in the DOM?

Event bubbling means that when an event happens on a child element, it first runs on that element and then moves up to its parent, then the grandparent and so on until it reaches the top.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, use document.createElement('tag') like document.createElement('div') then add it with parent.appendChild(element) to stick it inside another element. To remove it just use element.remove() or parent.removeChild(element) to target it through the parent.
